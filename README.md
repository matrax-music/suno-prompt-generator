# Suno Prompt Generator

**Professionelle Desktop-App für Suno AI Musik-Prompts**

Erstelle klanglich präzise Prompts aus über 100 Genres, 80+ Künstler-Einflüssen und Studio-Werkzeugen – per Mausklick.

## Features

- 🎵 **100+ Genre-Profile** mit maßgeschneiderten Parametern
- 🎤 **80+ Künstler-Datenbank** mit Klangbeschreibungen (auto-befüllt "Sounds like")
- 🎛️ **Studio-Modus**: Stem-Generator, Infill-Assistent (43 Templates), Remove FX, Bibliothek
- ✍️ **Create-Modus**: Umfangreiches Formular mit Stimmung, Tempo, Instrumente, Gesangsstil
- 🌐 **Zweisprachige UI** (DE/EN Toggle) – Prompt-Ausgabe immer auf Englisch
- 📊 **Qualitätsanalyse** mit Zeichenzahl-Warnungen
- ⭐ Favoriten, Verlauf, Template-Bibliothek, Batch-Generierung
- 💻 **Portabel** – keine Installation nötig (Windows 10/11)
