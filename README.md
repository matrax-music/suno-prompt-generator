# Suno Prompt Generator

Professionelle Desktop-App für Suno AI Musik-Prompts.

## Features
- 100+ Genre-Profile
- 80+ Künstler-Datenbank
- Studio-Werkzeuge (Stem-Generator, Infill-Assistent, Remove FX)
- Zweisprachige UI (DE/EN)
- Portabel – keine Installation nötig
